# Arena brand theme

Self-contained HTML bundle used as the source for the Arena Gamma theme.

- Fonts (Basel Grotesk, Basel Grotesk Mono, Martina Plantijn) inlined as base64.
- Colors, radii, shadow, and type scale lifted from `src/styles/index.css` in bd-prompt.
- Logo mark inlined as SVG (charcoal prompt-tile + ARENA eyebrow + wordmark).

Feed the raw URL of `index.html` to Gamma → *Create theme from URL* to import.
